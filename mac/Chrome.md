# Chrome

## Requirements

* [Homebrew](./Homebrew.md)

## Install (via brew)

```bash
brew install --cask google-chrome
```

## Setup

* Application > Chrome (open)
  * Open
* Use Chrome (default)
* Sign-in (optional) / Sync
* Resize
* Settings > Appearance > Page Zoom
    * 150%
* Dock
    * Keep in Dock
    * Reposition within Dock
