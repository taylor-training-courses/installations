# iTerm 2

## Requirements

* [Homebrew](./Homebrew.md)

## Install

```bash
brew install --cask iterm2
```

## Setup

* Applications: Search "iTerm"
  * iTerm
    * Open
* Reposition
* Dock
  * Keep in Dock
  * Remove normal Terminal app from Dock
  * Position within Dock

## Profile

* Settings > Window
  * Check "Remember the size and position"
* Settings > Profiles
* Default > Other Actions > Duplicate Profile
* Rename "Copy of Default" to "Training"
* Other Actions > Set as Default
* Training > Window
  * Cols 150
  * Rows 36
* Training > Text
  * Font: Menlo 26
  * Curser: vertical bar

## Verify

* Restart iTerm
* Resize / reposition to taste
