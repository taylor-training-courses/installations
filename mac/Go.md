# Installing Go

## Requirements

* [Homebrew](./Homebrew.md)

## Install via Homebrew

```bash
brew install go
```

Installs latest version.

### Test Versions

```bash
which go
go version
```

## VSCode Extension

* Open VSCode
* Go to Extensions. 
  * Type "Go" - choose the Go extension by Go Team at Google.
    * Install
* Close VSCode.
