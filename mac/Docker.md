# Docker

## Requirements

* [Command Line Tools](./README.md#command-line-tools)

## Installation

### Download

* Open browser
  * docker.com
* Download "Docker Desktop"
  * Download for Mac - Apple Silicon

### Docker Hub Shortcut

* Open new tab
  * hub.docker.com - copy URL
* Open new tab
  * Add shortcut
    * Name: Docker Hub
    * URL: (paste url) https://hub.docker.com

### Install

* Downloads > run Docker image
  * Drag "Docker" item into Applications
  * Eject "Docker"
* Applications > search "Docker"
  * Run Docker app
    * Open

### Setup

Opening Docker for first time will run setup wizard

* Docker Subscription Service Agreement
  * Accept
* Finish setting up Docker Desktop
  * Use recommended settings (password required)
    * Enter account password
  * A software update is required
    * Install Rosetta
      * Install
* Welcome to Docker
  * Change to Personal
  * Sign in with personal email
    * Continue with Google
    * Open Docker

### Dock

* Keep in Dock
* Move to dock position

### Verify

* Open Terminal

```bash
which docker
docker --version
docker run hello-world
```
