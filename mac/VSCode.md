# VSCode install

## Requirements

* [Homebrew](./Homebrew.md)

## Installation

### Install via Brew

```zsh
brew install --cask visual-studio-code
```

### Verify Installation

```zsh
which code
code --version
```

### First Open

```bash
code .
```

* Allow
* Dismiss first pages
* Resize / reposition to taste
* Select file to check file panel (.zprofile)
* Close / quit

### Dock

* Dock > Options > Keep in Dock
* Move to desired doc location

### Open from Applications

* Applications: Visual Studio Code
  * Open

## CoPilot

### Login to GitHub

* Open browser
  * Go to GitHub.com
* Login (or create account)
  * Verify with 2FA if needed

### Authorize VSCode

* Open VSCode
* Click on "CoPilot" icon
  * "Set up Copilot"
    * Sign in to use GitHub Copilot
      * Continue with GitHub
* Back to Browser
  * Select User to authorize (Back in browser)
    * Continue
    * Open Visual Studio Code (from browser)
      * Allow (if needed)

### Verify CoPilot

* CoPilot Chat
  * Ask mode
  * Choose "GPT-5 mini" (or any model)
  * "How are things going?" (any simple question to test AI integration)

## Cleanup

* Quit VSCode and Terminal app
