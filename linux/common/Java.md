# Java Installation

## Requirements

* Foundation for your distro
* Git
* SDKMan

## Installation

Open a terminal, use SDKMan to install the latest version of Java:

```bash
sdk install java
```

Restart terminal session.

### Verification

```bash
which java
java --version
```

## Visual Studio Code Integration

> Optional, but recommended if using VSCode for Java development

Open VSCode:

* Go to Extensions
* Search for "Java"
* Choose the "Java" extension by Oracle
