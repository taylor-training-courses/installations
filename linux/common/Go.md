# Go Install

## Requirements

If you haven't already, ensure you have followed the corresponding requirements before moving forward.

* Foundation:
    * [Debian-based Foundation](../debian/Foundation.md)
    * [RedHat-based Foundation](../redhat/Foundation.md)
* [Git](./Git.md)


## Installation

### Installation via Homebrew

```bash
brew install go
```

* Restart terminal

### Validation

```bash
which go
go version
```

## VSCode Integration

* Open VSCode
* Go to Extensions
* Search for "Go"
* Choose "Go" extension by Google
