# Development Setup on RedHat-family (or inspired) Distributions

Instructions are provided for those that prefer to do this themselves. The RedHat-family includes RedHat and any Linux distribution that is downstream of RedHat or inspired by Redhat and/or use DNF or YUM package managers - including Fedora, CentOS Stream, Bazzite, Rocky Linux, etc.

> Note: A user with __sudo__ is required for these instructions.

### Tested Ditributions and Versions

* Fedora 42
* CentOS 10 Stream

Should work with all recent versions of any RedHat-family distribution.
